# サンプル

AupDotNet を使用しているソフトウェアには以下のようなものがあります。

- [aup2exo](https://github.com/karoterra/aup2exo)
- [.aupExportTool](https://github.com/Share-aup/.aupExportTool)
- [AupFontExtractor](https://github.com/karoterra/AupFontExtractor)
- [AupRename](https://github.com/karoterra/AupRename)
- [AviUtlScriptExtractor](https://github.com/karoterra/AviUtlScriptExtractor)
- [IDExtractor](https://github.com/hoocheek/IDExtractor)

また、このリポジトリにも簡単な[使用例](https://github.com/karoterra/AupDotNet/tree/main/samples)があります。

| 名前 | 説明 |
|------|------|
| [ReadWriteAup][1] | aupファイルを読込みいくつかの情報を表示してaupファイルを保存する。 |
| [ReadWriteExEdit][2] | aupファイルを読込み拡張編集のいくつかの情報を表示してaupファイルを保存する。 |

[1]: https://github.com/karoterra/AupDotNet/tree/main/samples/ReadWriteAup
[2]: https://github.com/karoterra/AupDotNet/tree/main/samples/ReadWriteExEdit
