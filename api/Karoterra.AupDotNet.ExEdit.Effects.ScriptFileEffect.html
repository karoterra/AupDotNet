    Params

    スクリプトのパラメータ。 キーは変数名、値は変数の値です。

    Declaration
    public Dictionary<string, string> Params { get; set; }
    Property Value
    Type Description
    Dictionary<String, String>
    Remarks

    拡張編集のスクリプトファイルでは --param, --color, --file, --dialog によってパラメータを保存することができます。 これは拡張データに 変数名1=値1;変数名2=値2; のような形式で保存されます。 Params はこの変数名を Key、値を Value とするDictionary<TKey,TValue>です。 変数が local の場合は Key は local 変数名 になります。 拡張データに保存するときの最大バイト数は MaxParamsLength です。
