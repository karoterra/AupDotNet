    NameType

    マップの種類

    Declaration
    public FigureNameType NameType { get; }
    Property Value
    Type Description
    FigureNameType
    Remarks

    組込み図形以外のマップを使用している場合は Name にその情報が格納されます。 NameType は Name がどのような種類のマップの情報を格納しているかを表します。

    値説明
    BuiltIn 組込み図形が選択されています。 Name は null または空です。 FigureType を参照してください。
    Figure figureフォルダの画像が選択されています。 Name には画像のファイル名(拡張子無し)が入っています。
    File 外部ファイルが参照されています。 Name には画像ファイルの絶対パスの先頭に * が付いたものが入っています。 画像ファイルのパスを取得する場合は Filename を参照してください。
    Scene シーンが参照されています。 Name にはシーン番号の前に : が付いたものが入っています。 シーン番号を取得する場合は Filename を参照してください。
