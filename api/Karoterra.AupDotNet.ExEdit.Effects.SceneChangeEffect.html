    Remarks
    シーンチェンジの種類説明
    組み込みシーンチェンジ Name は空、 ScriptId にインデックスが入り、 Params はnull。
    exedit.scnのスクリプト Name にスクリプト名、 ScriptId に1が入る。
    ユーザ定義スクリプト Name にスクリプト名、 ScriptId に2が入る。
    transition画像 Name に画像ファイル名(拡張子無し)、 ScriptId に0が入り、 Params はnull。
